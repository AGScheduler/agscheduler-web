# agscheduler-web

> Web Client for AGScheduler using [Svelte](https://svelte.dev/) + [SvelteKit](https://kit.svelte.dev/) + [shadcn-svelte](https://shadcn-svelte.com/) + [Tailwind CSS](https://tailwindcss.com/).

## Developing

```bash
pnpm install

pnpm run dev
```

## Building

```bash
pnpm run build
```
